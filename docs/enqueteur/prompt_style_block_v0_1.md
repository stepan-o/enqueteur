# Enquêteur Prompt Style Block v0.1

Use this block as the default style instruction when generating or directing new Enquêteur visual assets.

---

## Core style block

Use the attached Enquêteur reference set as the primary visual canon:
- logo/title asset
- cast sheet
- MBAM museum map

The visual tone is:
- nocturnal
- theatrical
- cultured
- urban
- slightly magical

The target feeling is:
**Montréal at night under gallery lighting.**

Palette:
- deep indigo
- smoky plum
- aubergine
- petrol teal
- warm amber
- saffron
- ember coral
- vermilion
- parchment ivory

Rendering direction:
- painterly
- textured
- elegant
- emotionally readable
- warm light against cool midnight structure
- atmospheric but still gameplay-legible

Priority order:
1. readability
2. emotional tone
3. texture and atmosphere
4. decorative flourish

Avoid:
- cyberpunk neon
- flat casual mobile UI
- sterile corporate museum branding
- muddy brown realism
- fantasy parchment ornament
- cute detective cartoon energy
- horror-goth exaggeration

The work should feel like:
**museum-night mystery + painterly human drama + graphic wayfinding intelligence**

---

## UI add-on block

For UI or shell work:
- make it feel like gallery signage meets detective notebook
- preserve clear hierarchy and bilingual readability
- use dark cool surfaces with warm focal accents
- keep ornament restrained and intelligent
- prioritize legibility over texture density

---

## Portrait add-on block

For character portraits or state-card art:
- expressive, painterly, psychologically readable faces
- warm/cool contrast required
- subdued background fields
- emotional specificity without melodrama
- realistic proportions, no plastic cinematic gloss

---

## Environment add-on block

For room, map, or object presentation work:
- architectural and civic rather than fantastical
- museum, archive, and institutional atmosphere
- route/signage logic can inform composition
- warm light reveals what matters
- environment must support investigation clarity first

---

## Motion add-on block

For motion/FX direction:
- subtle drift, shimmer, glow, dust, and restrained pulses
- no arcade bounce
- no noisy particle spam
- no constant flashy animation
